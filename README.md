# Crime-Data-Analysis

# Overview
This project analyzes crime data to identify patterns in crime types, locations, and time trends.

# Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

# Analysis Performed
- Data cleaning and preprocessing
- Crime type distribution
- Area-wise crime analysis
- Time-based trend analysis

# Key Insights
- Vehicle theft is one of the most common crimes
- Certain areas show higher crime frequency
- Crime occurrences vary across months and days

# Conclusion
This project demonstrates how data analysis can be used to extract meaningful insights from large datasets.
